# 创建节点池

调用CreateClusterNodePool为集群创建节点池。

## 调试

[您可以在OpenAPI Explorer中直接运行该接口，免去您计算签名的困扰。运行成功后，OpenAPI Explorer可以自动生成SDK代码示例。](https://api.aliyun.com/#product=CS&api=CreateClusterNodePool&type=ROA&version=2015-12-15)
