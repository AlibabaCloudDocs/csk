调用DeleteClusterNodepool删除节点池。

## 请求头

该接口使用公共请求头，无特殊请求头。更多信息，请参见[公共请求参数](~~167755~~)。

## 请求语法

```
DELETE /clusters/{ClusterId}/nodepools/{NodepoolId} 
```
