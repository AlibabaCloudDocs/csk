# 查看集群详细信息

通过执行arc-cs cluster get <clusterId\>命令查看您在容器服务中创建的集群详细信息。

关于该命令的详细参数说明，请参见API文档[查询集群实例](/cn.zh-CN/API参考/集群/查询集群实例.md)。

## CLI示例

请求示例：

```
arc-cs cluster get cbfa079eb225c44e7bce86c626f0c****
```
